# Quick Start Guide for Zhong Boss Clothing Factory Game

## Getting Started

Welcome to the Zhong Boss Clothing Factory! Follow the instructions below to get your game up and running.

### Prerequisites
- Make sure you have the latest version of Python installed on your machine.
- Install required dependencies by running:
  ```bash
  pip install -r requirements.txt
  ```

### Starting the Game
1. Clone the repository:
   ```bash
   git clone https://github.com/<owner>/zhong-boss-clothing-factory.git
   ```
2. Navigate into the project directory:
   ```bash
   cd zhong-boss-clothing-factory
   ```
3. Run the game:
   ```bash
   python main.py
   ```

### Additional Notes
- For further customization and settings adjustment, refer to the `config.py` file.
- Check the `docs` folder for additional features and details.

Happy gaming!
